# 🗳️ Python Election Data Analysis Project

## 📌 Project Overview

This project demonstrates how to perform **data cleaning, preprocessing, exploratory data analysis (EDA), and advanced analysis** on election datasets using **Python** and **Pandas**.

The project uses three CSV files representing **Candidates**, **Constituencies**, and **Election Results**. The datasets intentionally include duplicate records and missing values to simulate real-world data and practice data cleaning techniques.

---

## 📂 Dataset

The project consists of the following datasets:

* **candidates.csv**

  * Candidate ID
  * Candidate Name
  * Party

* **constituencies.csv**

  * Constituency ID
  * Constituency Name
  * State

* **results.csv**

  * Result ID
  * Candidate ID
  * Constituency ID
  * Votes
